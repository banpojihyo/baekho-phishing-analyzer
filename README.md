# baekho-phishing-analyzer

공개용으로 정리한 `Baekho` 피싱 분석기 저장소입니다.

이 저장소는 private 원본 저장소에서 아래 범위만 선별한 public showcase/export 세트입니다.

## 포함 범위

- FastAPI 앱과 핵심 분석 로직
- `baekho.app` 랜딩/데모/제보 페이지 정적 자산
- 공개 가능한 테스트 일부
- 공개 가능한 프로젝트/제보 관련 문서 일부

## 제외 범위

- 대화 로그
- 배포/운영 런북
- AWS/서버 운영 관련 설정
- 내부 발표 자료, 민감한 운영성 스크립트, 데이터셋/로그 디렉터리

## 공개 목적

이 저장소는 `baekho.app`에서 소개하는 피싱 분석 MVP의 공개 가능한 코드와 문서를 보여주기 위한 저장소입니다.
운영 환경, 내부 데이터, 배포 절차, 민감한 작업 기록은 포함하지 않습니다.

## 참고

- 현재 public 사이트는 `baekho.app`
- 공개 코드 구조는 `docs/codebase-map.md`에 정리되어 있습니다.
